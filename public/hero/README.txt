OBJET 3D DU HERO — spécifications d'export
==========================================

Dépose ici ton rendu 3D (Spline / Blender / IA). Puis préviens Claude,
qui le branchera dans le hero à la place du blob CSS actuel.

NOMMAGE ATTENDU (choisis selon ce que tu exportes) :
  - Image  : object.png        (fond TRANSPARENT)
  - Vidéo  : object.webm        (idéalement avec transparence/alpha)
             ou object.mp4      (si pas de webm)

SPECS RECOMMANDÉES
------------------
Format cible : le fond du site est NOIR (#050505).

IMAGE (le plus simple) :
  - PNG, fond TRANSPARENT (pas de fond noir « collé »)
  - Carré, haute résolution : 1600 x 1600 px (ou 2000 x 2000)
  - Objet centré, avec un peu de marge autour
  - Poids : vise < 800 Ko (compresse via tinypng.com si besoin)

VIDÉO (si objet animé / qui tourne) :
  - WebM avec alpha si possible, sinon MP4 (H.264)
  - Carré ~1200 x 1200, boucle courte (4–8 s), 30 fps
  - Muette, pensée pour tourner en boucle
  - Poids : vise < 3 Mo

RENDU / MATÉRIAU (pour coller au thème)
---------------------------------------
  - Matériau : verre sombre / metal liquide, très réfléchissant
    (Metallic élevé, Roughness basse)
  - Éclairage doux en haut à gauche → beau reflet spéculaire
  - Une pointe de reflet iridescent (violet/bleu) discret, optionnel
  - Fond transparent (surtout PAS un fond noir opaque)

Une fois le fichier ici : dis « c'est bon, objet dans public/hero »,
et je le branche en 30 secondes (fondu propre, fallback automatique).
